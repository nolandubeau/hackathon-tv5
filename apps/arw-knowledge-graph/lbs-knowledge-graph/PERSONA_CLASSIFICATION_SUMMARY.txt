TARGETS RELATIONSHIP SCHEMA
================================================================================

Properties:
  - relationship_type: "TARGETS"
  - persona_id: persona identifier
  - relevance: 0-1 score (threshold: 0.6)
  - is_primary: boolean (highest relevance)
  - journey_stage: awareness|consideration|decision|action|retention
  - signals: list of targeting indicators
  - intent: why this targets this persona
  - confidence: 0-1 score
  - extracted_by: model name (gpt-4o-mini)

Example:
  Page → TARGETS → Persona:persona_prospective
    relevance: 0.85
    is_primary: true
    journey_stage: consideration
    signals: ["MBA programme", "career switch", "application process"]
